# Prism Theme One Dark

PrismJS theme loosely based on the [Atom's One Dark theme][atomonedark].

> Originally forked from: [AGMStudio][agmstudio] 

## Install

1. Install [prismJS][prismjs].
2. Use this `prism.css`.
